# Layer 7 — The Application Layer

**Purpose:**  
The Application Layer interacts directly with **end users** and provides the services applications need to communicate over a network.

---

## What Happens at This Layer?
- Provides network services to software applications
- Handles HTTP requests for web browsing
- Manages email, file transfer, and remote login services
- Enables user authentication and permissions

---

## Real-World Examples
- Web browsers (HTTP/HTTPS)
- Email (SMTP, IMAP, POP3)
- DNS (domain name lookup)
- FTP, SFTP
- Chat applications

---

## Summary
Layer 7 is the closest to the user and provides the **actual tools and services** used in network communication.
